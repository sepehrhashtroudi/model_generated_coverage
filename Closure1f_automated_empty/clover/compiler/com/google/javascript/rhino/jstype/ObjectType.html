54    /**
55    * Object type.
56    *
57    * In JavaScript, all object types have properties, and each of those
58    * properties has a type. Property types may be DECLARED, INFERRED, or
59    * UNKNOWN.
60    *
61    * DECLARED properties have an explicit type annotation, as in:
62    * <code>
63    * /xx @type {number} x/
64    * Foo.prototype.bar = 1;
65    * </code>
66    * This property may only hold number values, and an assignment to any
67    * other type of value is an error.
68    *
69    * INFERRED properties do not have an explicit type annotation. Rather,
70    * we try to find all the possible types that this property can hold.
71    * <code>
72    * Foo.prototype.bar = 1;
73    * </code>
74    * If the programmer assigns other types of values to this property,
75    * the property will take on the union of all these types.
76    *
77    * UNKNOWN properties are properties on the UNKNOWN type. The UNKNOWN
78    * type has all properties, but we do not know whether they are
79    * declared or inferred.
80    *
81    */
