56    /**
57    * The {@code UnionType} implements a common JavaScript idiom in which the
58    * code is specifically designed to work with multiple input types. Because
59    * JavaScript always knows the run-time type of an object value, this is safer
60    * than a C union.<p>
61    *
62    * For instance, values of the union type {@code (String,boolean)} can be of
63    * type {@code String} or of type {@code boolean}. The commutativity of the
64    * statement is captured by making {@code (String,boolean)} and
65    * {@code (boolean,String)} equal.<p>
66    *
67    * The implementation of this class prevents the creation of nested
68    * unions.<p>
