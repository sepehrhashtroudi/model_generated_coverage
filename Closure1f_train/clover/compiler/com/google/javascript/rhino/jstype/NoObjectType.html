48    *
49    * Although JavaScript programmers can't explicitly denote the bottom
50    * Object type, it comes up in static analysis. For example, if we have:
51    * <code>
52    * var x = function() {};

54    * f(x);
55    * }
56    * </code>
57    * We need to be able to assign {@code x} a type within the {@code f(x)}
58    * call. It has no possible type, but {@code x} would not be legal if f
59    * expected a string. So we assign it the {@code NoObjectType}.
60    *
61    * @see <a href="http://en.wikipedia.org/wiki/Bottom_type">Bottom types</a>
